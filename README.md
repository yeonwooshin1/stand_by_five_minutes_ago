## ✨ 주요 기능 및 담당 역할

*   **전체:** 프로젝트 실행 환경 구축 및 총괄
*   **신연우:**
    *   사용자 인터페이스 (UI/UX)
    *   JJWT 라이브러리를 활용한 인증 토큰 보안
    *   메일 발송 기능 (JavaMail)
*   **옹태경:**
    *   역할 템플릿 관리
    *   Apache POI 라이브러리를 활용한 데이터 Excel Export
*   **최동진:**
    *   업무 체크리스트
    *   OpenPDF 라이브러리를 활용한 데이터 PDF 출력
    *   GraphHopper 및 대중교통 API를 활용한 출장 인력 이동 경로 제공
